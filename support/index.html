    Supported Parts

    This is a (mostly complete) list of boards and other components supported by OmniBox, along with the necessary STL. The linked git folders should contain STEP and Fusion files to allow modification. Please open an issue if any are missing.

    Trays

    MCU Trays

    These parts are in the Trays\MCU git folder. Each component has its own subfolder.

    Component Note
    BTT Octopus
    BTT SKR 1.3+ Compatible with all SKRs 1.3 and newer.
    BTT SKR Mini E3 Compatible with 1.x and 2.
    Creality boards Use the BTT SKR Mini E3 mount.

    CPU Trays

    These parts are in the Trays\CPU git folder.

    Component STL
    Raspberry Pi 3B+ CPU Tray - RPi 3B Plus.stl
    Raspberry Pi 4B CPU Tray - RPi 4B.stl

    Lower Bay Trays

    The files for these trays are stored in the Trays\Lower Bay git folder.

    Some trays include space for 40x40x10mm fans. This is specified in the file name.

    Component STL Note
    Generic LM2596 Lower Bay Tray - 40mm Fan and Two Bucks v2.stl Typical current limit of 2.5A max.
    DROK LM2596 with LED Lower Bay Tray - LM2596 with LED.stl Current limit is 3A max, 2A continuous.

    Panels

    Display Screen Panels

    Each display screen has a subfolder in the Panels/Display git folder. There is an optional STL for a display knob, as well.

    Component Folder Notes
    Creality 12864 Stock Generic 12864 Also known as CR-10 stock display. Ender 3 V2 display not supported.
    FYSETC Mini 12864 Mini 12864 This includes BTT and other clones.
    BTT 2.4" TFT BTT 2.4 TFT This product is discontinued, but I still use it, so there's a part.
    BTT 3.5" TFT BTT 3.5 TFT

    Front and Rear Panels

    Both of these components are highly customizable, so this list may be incomplete.

    Part or Connector Panel Location Notes
    MicroSD panel mount extension Front
    USB C panel mount extension Front or Rear
    JST SM panel mount connectors Rear 2-5 pins
    Molex Micro Fit 3.0 connectors Rear Currently only supported in the 2-pin, 2-row configuration.
