# Trays/MCU/BTT SKR

## Overview

This tray mounts a BIGTREETECH SKR 3 EZ board. It is *not* compatible with the regular SKR 3.

| Field                 | Description               |
|-----------------------|---------------------------|
| Category              | MCU Tray                  |
| Product Supported     | BIGTREETECH SKR 3 EZ 		|
| STEP File             | Yes                       |
| Fusion 360 Archive    | No                        |
| Version Added         | 0.9.9                     |
